Derivational morphology

Adjective → adverb = If starts with vowel: Suffix -liχ
Else: Suffix -aliχ
Adjective = If ends with vowel: Suffix -ɾɪ
Else: Suffix -iːɾɪ
Adjective → noun (the quality of being [adj]) = If ends with vowel: Suffix -rəu
Else: Suffix -iːrəu
Noun → adjective (having the quality of [noun]) = If starts with vowel: Suffix -piː
Else: Suffix -viːɛ
Adjective → verb (to make something [adj]) = If ends with vowel: Suffix -s
Else: Suffix -iːs
Noun → adjective relating to noun (e.g. economy → economic) = Suffix -bɾeːɡ
One who [verb]s (e.g. paint → painter) = If ends with vowel: Suffix -si ENLE -iːsi
Verb → noun (the act of [verb]) = Suffix -əuχt
Femenize = If starts with vowel: Suffix -ʃ
Else: Suffix -aʃ
Place of (e.g. wine → winery) = Suffix -iː
Diminutive = If starts with vowel: Suffix -ɟɪlaː
Else: Suffix -ɪlaː
Augmentative = Suffix -iːχt
Tending to = If ends with vowel: Suffix -ʃt
Else: Suffix -aʃt
Verb → adjective (result of doing [verb]) = If starts with vowel: Prefix l-
Else: Prefix lɛ-
Verb → noun that verb produces (e.g. know → knowledge) = Suffix -ɔnˠ
Of the hills = Suffix -ʃɪdɛ
Noun to verb = If starts with vowel: Suffix -n
Else: Suffix -en
To language = If starts with vowel: Suffix -lɡɛ
Else: Suffix -ɪlɡɛ
Verb to past perf = Prefix ɟe-
